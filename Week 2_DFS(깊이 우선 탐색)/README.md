## 2주차_알고리즘 스터디 DFS(깊이 우선 탐색)

2주차는 DFS(깊이 우선 탐색) 알고리즘에 관련된 문제들입니다.

본인이 소지하고 있는 알고리즘 학습 서적과 구글링을 통하여 학습 한 뒤, 문제를 풀이해주시면 되겠습니다.

다음주는 BFS 문제가 과제로 출제가 될 예정이니 같은 공부하시면 더 좋을 것 이라고 생각합니다.
